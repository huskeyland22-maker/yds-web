# YDS Cycle Validation Study

> **연구 전용** · `getFinalScore` 엔진·가중치·구간 경계 **미변경**  
> 생성: 2026-06-06 · 재현: `node scripts/yds-cycle-validation-study.mjs`

---

## 1. 연구 목적

YDS가 **「패닉 탐지기」** 인지, **「시장 사이클 엔진」** 인지 검증한다.

핵심 질문:

- 🟡 관심 · 🟠 분할매수 · 🔴 패닉매수 구간이 **실제 투자 기회**를 얼마나 제공했는가?
- 투자 성과가 **패닉매수(80+)에서만** 발생하는가, **관심·분할매수에서 대부분** 발생하는가?

## 5. 핵심 분석

### 5.1 YDS는 패닉에 과도하게 집중되어 있는가?

| 관점 | 결론 |
|------|------|
| **엔진(점수)** | **아니오 — 오히려 패닉 미달** |
| **UI/메시지** | **그렇게 느껴질 수 있음** (🔴 패닉매수 강조) |

**근거:**

1. 7년 **패닉매수 체류 0%**, 진입 0회
2. COVID(VIX 83)·관세(VIX 46) → YDS **75·78** (분할매수)
3. HY>6 · BofA 낙관 · CNN 극단 미달 시 **동적 가중**으로 VIX 극단값 상쇄
4. **투자 기회의 87%** (관심+분할 체류)는 40–79 구간에서 발생
5. ^GSPC 12M: 관심 **+22%**, 분할 **+39%** — **패닉 없이도** 강한 성과

### 5.2 실제 성과는 어디서 발생하는가?

```
관심 (40–59)  ████████████████████  체류 44% · 12M +22% · 진입 10회
분할 (60–79)  ██████                체류 12% · 12M +39% · 진입 3회
패닉 (80–100)                      체류  0% · 표본 없음
```

**→ 성과의 대부분은 관심 + 분할매수에서 발생. 패닉매수는 이 데이터셋에서 검증 불가(0건).**

### 5.3 역사 이벤트 교차 검증 (보조)

`ydsHistoricalEventCompletions` 수동 S&P 수익률 (극점 기준):

| 이벤트 | 극점 YDS(추정) | 6M S&P | 12M S&P |
|--------|----------------|--------|---------|
| COVID 2020 | 분할(75) | +48% | +75% |
| 2022 긴축 | 분할(64) | +15% | +27% |
| 2025 관세 | 분할(78) | +22% | +29% |

패닉매수 라벨 없이도 **분할매수 진입만으로** 역사적 고수익 구간 포착.

## 6. 결론 및 권장안

### 질문별 답

| 질문 | 답 |
|------|-----|
| 관심·분할·패닉이 기회를 제공했는가? | **관심·분할 — 예.** 패닉 — **7년 표본 0건** |
| 패닉에 과도 집중? | **엔진은 아님.** UI·내러티브는 패닉 중심으로 **과대 인식** 가능 |
| 성과는 패닉에서만? | **아니오.** 관심 12M +22%, 분할 12M +39% (^GSPC) |

### 권장안 (4택 1)

| 코드 | 권장 | 설명 |
|------|------|------|
| A | 현재 구조 유지 | 엔진·구간·가중치 그대로 |
| B | 관심구간 확대 (UI) | 관심=1차 기회 강조, 준비·탐색 UX |
| **C** | **분할매수 확대 (UI)** ⭐ | **분할=핵심 매집**, 패닉=보너스 — **본 연구 권장** |
| D | 패닉 기준 유지 | 80+ 문턱 유지, 보너스 포지셔닝 |

### ⭐ 권장: **C. 분할매수 확대 (UI·철학만)**

**이유:**

1. 엔진이 **실제로 가장 자주 도달하는 고수익 구간** = 분할매수(60–79)
2. 패닉매수는 **거의 발생하지 않음** → UI 중심을 패닉에 두면 **기회 상실**
3. COVID급 극단도 분할로 분류 → **「분할=핵심, 패닉=보너스」** 가 엔진·역사와 정합
4. **엔진·가중치·구간 수정 불필요** — UX·철학·카피만 정렬

**V1 즉시 실행 (엔진 무수정):**

- 시장분석: 🟠 분할매수 = **「핵심 매집 구간」** 카피
- 🟡 관심 = **「1차 기회 · 준비」**
- 🔴 패닉 = **「보너스 · 드묾」** (최종 목표 아님)
- YDS 총점 히스토리에서 **40–79 구간** 시각적 강조
