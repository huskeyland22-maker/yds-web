## 1. 검토 배경

### 1.1 문제 제기

현재 🟡 **관심구간**(40–59)은 이름만 보면 **「그냥 관심」** — 수동적·느슨한 인상을 준다.  
최근 연구는 실제 역할이 다르다고 보여준다.

| 연구 사실 | 함의 |
|-----------|------|
| 관심 체류 **43.9%** | YDS 시간의 거의 절반 — **핵심 사이클 단계** |
| 관심 진입 → eventual 분할 **90%** | 대부분 **분할매수 퍼널의 상류** |
| 관심→분할 중앙 소요 **~437일** | 단기 스파이크가 아니라 **장기 매수 준비** |
| 관심 12M ^GSPC **+22.4%** | 「관심」이 아니라 **실질적 1차 기회** |

실제 행동은 다음에 가깝다.

- 매수 **준비** · 종목 **발굴** · **현금** 확보 · **1차 진입** 검토

현재 UI 철학(`ydsCyclePhilosophy.js`)은 이미 **「1차 기회 · 좋은 기업 탐색」** 으로 보정 중이나, **공식 구간명 「관심구간」** 은 여전히 구(舊) 톤을 끌고 온다.

### 1.2 검토 범위

| 포함 | 제외 |
|------|------|
| 🟡 40–59 **표시 명칭** · 서브카피 · 태그라인 정합 | 점수·구간 경계·가중치 |
| 5단계 **내러티브 흐름** | `interest` 등 **엔진 ID** |
| 철학 **「분할=실행 · 패닉=보너스」** 와 정합 | 코드 일괄 반영 (본 문서는 권장만) |

### 1.3 현재 5단계 (As-Is)

```
🔵 과열구간 ──→ 🟢 중립구간 ──→ 🟡 관심구간 ──→ 🟠 분할매수 ──→ 🔴 패닉매수
  리스크 관리      관찰·준비         ???            핵심 실행         보너스
```

**공식 라벨** (`panicMacroV1Status.js`): `관심구간`  
**철학 role** (`ydsCyclePhilosophy.js`): `1차 기회`  
**태그라인**: 「**관심**에서 준비하고, 분할매수에서 실행한다.」

→ **명칭(관심) vs 역할(준비·1차 기회) 불일치** 가 UX 핵심 이슈.

## 2. 후보별 장단점

### A. 🟡 관심구간 (현행 유지)

| 장점 | 단점 |
|------|------|
| 기존 사용자·문서·코드와 **100% 일치** — 마이그레이션 비용 0 | 「관심」= 수동적 — **44% 체류·90% 퍼널** 과 역할 불일치 |
| 짧고 발음 쉬움 (2음절) | **관심종목**(Watchlist)과 혼동 — 「구간 vs 기능」 |
| 태그라인·레일 라벨 변경 불필요 | 중립(「관심 종목 정리」)과 **행동 차이가 이름으로 안 드러남** |
| | 연구 결론 「관심에서 준비」는 **카피로만 보정** — 구간명은 여전히 약함 |

**한 줄:** 안전하지만, **연구가 드러낸 「준비 퍼널」 메시지를 이름만으로는 전달 못함.**

---

### B. 🟡 준비구간

**서브카피 예:** 매수 준비 · 종목 발굴 · 현금 확보

| 장점 | 단점 |
|------|------|
| 연구·철학과 **직접 정합** — 「관심에서 **준비**」의 행동을 이름에 반영 | 🟢 중립 introLine **「관찰 및 준비」** 와 **「준비」 키워드 충돌** |
| 분할매수(실행)로 가는 **사다리** 역할이 명확 | 태그라인 「관심에서 준비」→ **「준비에서 준비」** 중복 — 문구 개정 필요 |
| 14개월급 **장기 준비** 구간과 어울림 | 「준비」만으로는 **매수 vs 현금** 방향성이 다소 모호 |
| | 과열·중립 대비 **행동 강도**는 올라가지만, 「기회」 감은 약함 |

**완화:** 중립 카피를 **「관찰·모니터링」** 으로, 🟡만 **「준비」** 독점 — companion UX (엔진 무관).

---

### C. 🟡 매수준비구간

**서브카피 예:** 분할매수 직전 단계

| 장점 | 단점 |
|------|------|
| **매수** 의도가 가장 분명 — 행동 유도력 높음 | **6글자** — 모바일 레일·차트 범례에 **과장** |
| 「분할 직전」으로 🟠와 **인과 연결** | 중앙값 **437일** — 「직전」은 **단기 승격(5일)** 에만 맞고 **대부분과 불일치** |
| | 「매수준비」= 곧 사라? **조급함·FOMO** 유발 위험 |
| | 중립·관심 경계(40)에서 **갑작스럽게 매수** 느낌 |

**한 줄:** 의도는 좋으나 **길이·「직전」 부정확·조급함** — 단독 권장 어려움.

---

### D. 🟡 기회구간

**서브카피 예:** 1차 기회 포착 · 우량주 선별

| 장점 | 단점 |
|------|------|
| Cycle Validation **+22% 12M** 과 **「기회」** 내러티브 일치 | Philosophy §8.2 이미 **40–79 = 「기회 구간」** 보조 텍스트 — 🟡만 쓰면 **🟠와 중복** |
| 「관심」보다 **적극적** — 패닉 대기를 깨는 톤 | 🟢 중립도 장기적으로는 기회 **전 단계** — 경계 모호 |
| 5단계 흐름: 과열(위험) → … → **기회** → **실행(분할)** → 보너스 | 「기회」는 **무엇을 할지**(준비·현금·탐색)보다 **결과** 강조 |
| 짧음 (3음절) | ETF·일반 「투자 기회」와 **범용어** — YDS 특유성 약함 |

**한 줄:** 수익·사이클 스토리와 잘 맞으나, **🟠와의 차별·행동 가이드**는 서브카피 필수.

---

### E. 🟡 탐색구간

**서브카피 예:** 좋은 기업 탐색 · 종목 발굴

| 장점 | 단점 |
|------|------|
| `ydsCyclePhilosophy` **「좋은 기업 탐색」** 과 일치 | 🟢 중립 **「종목 탐색 우선」** 과 **거의 동일** — 단계 구분 실패 |
| Research·Stock Radar **제품 동선**과 연결 용이 | **44% 체류·90% DCA 퍼널** — 탐색만으로는 **현금·1차 진입** 누락 |
| 부드러운 톤 — 초보자 부담 적음 | **행동 유도력 최약** — 「구경」 구간으로 오해 |
| | 분할매수 **핵심 실행** 철학과 비교 시 **너무 소극적** |

**한 줄:** 중립과 **역할 겹침** — 5단계 사다리에서 **승격 동기 부족**.

## 3. 사용자 이해도 (5점 만점)

| 후보 | 직관성 | 5단계 구분 | Watchlist 혼동 | 중립과 구분 | **합계** |
|------|--------|------------|----------------|-------------|----------|
| A 관심 | 4 | 2 | 2 | 2 | **10/20** |
| B 준비 | 4 | 4 | 4 | 3* | **15/20** |
| C 매수준비 | 3 | 4 | 4 | 4 | **15/20** |
| D 기회 | 4 | 3 | 4 | 3 | **14/20** |
| E 탐색 | 3 | 2 | 3 | 1 | **9/20** |

\* B는 **중립 카피에서 「준비」 제거** 시 4점 → **16/20**.

**신규 사용자 10초 테스트 (가상):**

- 「YDS 47점」→ A: *「관심이네, 뭐 볼까?」* / B: *「준비하라는 거구나」* / C: *「곧 사?」* / D: *「기회구간이네」* / E: *「찾아보라는 거? 중립이랑 뭐가 달라?」*

**→ B·D가 「사이클 단계」로 가장 잘 읽힘. E·A는 중립과 붙어 보임.**

## 4. 투자 행동 유도력

철학 목표: **패닉(80+) 대기 금지** · **관심 퍼널에서 미리 준비** · **분할에서 실행**.

| 후보 | 유도 행동 | 패닉 대기 억제 | 분할 전 준비 강화 | FOMO/과잉매매 |
|------|-----------|----------------|-------------------|---------------|
| A 관심 | 낮음 — 관망·워치리스트 | 중 | 중 | 낮음 |
| **B 준비** | **높음** — 현금·리스트·소량 검토 | **높음** | **높음** | 낮음 |
| C 매수준비 | 높음 — 즉시 매수 압력 | 중 | 중 | **높음** |
| D 기회 | 중–높음 — 기회 인식 | 높음 | 중 | 중 |
| E 탐색 | 낮음 — 리서치만 | 낮음 | 낮음 | 낮음 |

**연구 앵커:** 관심 10회 중 **직접 분할 전환 20%** — 대부분은 **수개월 준비 후** 분할.  
→ **「준비」** 는 적합, **「직전·곧 매수」** 는 부적합.

## 5. YDS 철학 적합도

### 5.1 이상적 5단계 내러티브

```
🔵 리스크 관리  →  🟢 관찰  →  🟡 ???  →  🟠 핵심 실행  →  🔴 보너스
   (줄인다)        (본다)      (쌓는다)     (분할매수)       (드묾)
```

| 후보 | ??? 슬롯 적합 | 분할=핵심 실행 | 패닉=보너스 | 태그라인 호환 |
|------|---------------|----------------|-------------|---------------|
| A 관심 | △ (역할≠이름) | ○ | ○ | ○ (현행) |
| **B 준비** | **◎** | **◎** | ○ | △ (태그라인 수정) |
| C 매수준비 | ○ | △ (분할과 역할 겹침) | ○ | △ |
| D 기회 | ○ | ○ (기회→실행) | ○ | △ |
| E 탐색 | △ | △ | ○ | △ |

### 5.2 태그라인 정합 (To-Be 예시)

| 현행 | B 채택 시 | D 채택 시 |
|------|-----------|-----------|
| 관심에서 준비하고, 분할매수에서 실행한다. | **준비구간에서 쌓고, 분할매수에서 실행한다.** | **기회를 포착하고, 분할매수에서 실행한다.** |

서브: *「패닉을 기다리는 시스템이 아닙니다. 준비·분할매수가 실전 기회의 중심입니다.」*

## 6. 5단계 흐름 — 후보별 전체 맥락

### 6.1 권장 조합 (B 채택 + 중립 카피 미세 조정)

```
🔵 과열구간     🟢 중립구간        🟡 준비구간         🟠 분할매수      🔴 패닉매수
리스크 관리  →  관찰·모니터링  →  매수 준비·발굴  →  핵심 매집·실행  →  보너스·극단
현금 확보       종목 리스트       현금·1차 검토       비중 확대         드묾
```

### 6.2 D 채택 시

```
… → 🟡 기회구간 (1차) → 🟠 분할매수 (핵심) → 🔴 패닉매수 (보너스)
```

40–79 보조 문구 「기회 구간」은 **🟡=1차 · 🟠=핵심 실행** 으로 **이중 정의** 필요.

## 7. 최종 권장안

### 7.1 1순위: **B — 🟡 준비구간** (+ 서브카피)

| 항목 | 내용 |
|------|------|
| **공식 명칭** | 🟡 **준비구간** (40–59) |
| **서브카피** | 1차 기회 · 종목 발굴 · 현금 확보 · 소량 진입 검토 |
| **role (철학)** | `매수 준비` (현 `1차 기회` 유지 가능) |
| **엔진 ID** | `interest` **유지** (코드·API 불변) |

**이유 요약**

1. Opportunity Frequency **90% DCA 퍼널 · 44% 체류** = 「준비」가 본질  
2. 태그라인 「…에서 준비하고」와 **행동 동사 정렬** (명칭만 바꿔도 스토리 일관)  
3. 🟠 **분할매수 = 실행** · 🔴 **패닉 = 보너스** 와 **사다리 구조** 명확  
4. C보다 **덜 조급**, E·A보다 **적극적**, D보다 **행동 지침**이 분명  

**필수 companion (엔진 무관 UX)**

- 🟢 중립: 「관찰 및 **준비**」→「**관찰·모니터링**」 등 — **「준비」는 🟡 전용**  
- 태그라인: **「준비구간에서 쌓고, 분할매수에서 실행한다. 패닉매수는 보너스다.」**  
- 레일: `🔵 과열 · 🟢 중립 · 🟡 준비 · 🟠 분할매수 · 🔴 패닉매수`

### 7.2 2순위: **D — 🟡 기회구간**

- **수익·1차 기회** 강조가 필요할 때 (온�oarding·마케팅)  
- 단, 🟠와 **「기회 vs 실행」** 이중 레이블 가이드 필수  

### 7.3 비권장

| 후보 | 이유 |
|------|------|
| **A 유지** | 연구·철학 보정 후에도 **명칭–역할 gap** 잔존 — Watchlist 혼동 지속 |
| **C 매수준비** | 「직전」 부정확 · UI 길이 · FOMO |
| **E 탐색** | 중립과 **역할 중복** · 행동 유도력 최약 |

### 7.4 절충안 (마이그레이션 완화)

단계적 rollout:

1. **Phase 1:** 공식 라벨 `관심구간` 유지, UI에 **「준비구간 (관심)」** 병기 1–2 릴리스  
2. **Phase 2:** `준비구간` 단독 + 구명 「관심」은 FAQ·릴리스 노트에만
