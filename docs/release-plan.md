## 배포 원칙

- 작은 UI 수정마다 **즉시 배포하지 않음**
- Release Candidate 항목이 **5~10개 이상** 누적되면 배포
- 배포 전 `docs/deploy-checklist.md` 전 항목 확인
- RC 검증: `VITE_APP_RELEASE_CHANNEL=rc npm run build` 후 preview·스테이징 확인

---

## RC 로컬 실행 예시

```bash
cd vite-project
# 개발 (배지 DEV)
npm run dev

# RC 빌드 미리보기 (배지 RC)
set VITE_APP_RELEASE_CHANNEL=rc
npm run build
npm run preview
```
